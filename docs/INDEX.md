## ⚡ Quick Commands

```bash
# Build and run
Ctrl+Shift+B → Build & Run Debug

# Debug the app
F5 → Debug LibMan (GDB)

# Run tests
Ctrl+Shift+B → Tests: Run All Tests

# Quick help
Ctrl+Shift+P → "Tasks: Run Task"
```
